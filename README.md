# DevPath.io - Interactive Fullstack Developer Roadmap

**DevPath.io** is a premium, highly interactive, and responsive web application designed to guide developers through their learning journey from basic web foundations to advanced system design. 

This roadmap features a curated path highlighting modern industry-standard technologies such as **Next.js**, **FastAPI**, **Docker**, and **Kubernetes**, alongside core concepts, references, mini-quizzes, and custom project milestones.

---

## Key Features

- **Interactive Node Tree**: Click on nodes to display detailed guidelines, core checklists, and curated resources in a slide-out panel.
- **Career Path Filtering**: Toggle between specialized paths (**Frontend Specialist**, **Backend Specialist**, **DevOps & Cloud**, or **Fullstack Generalist**) to automatically fade out non-relevant nodes and highlight your goals.
- **Active Search**: Instantly find nodes, frameworks, languages, or concepts using the real-time header search bar.
- **Visual Progress Tracker**: A dynamic sidebar containing a circular progress ring and metrics tracking total nodes completed and quizzes passed. Progress is persisted locally using `localStorage`.
- **Dynamic Dependency Mapping**: Curved vector connections are rendered in real time over an SVG canvas to represent prerequisites and learning dependencies.
- **Interactive Mini-Quizzes**: Test your knowledge on each node with 3 multiple-choice questions. Scoring 100% automatically completes the node and updates your profile metrics.
- **Light & Dark Theme Switcher**: Easily switch between a cyber-neon dark mode and a sleek, high-contrast light mode with a single header button.
- **Milestone Badges**: Earn specific visual badges for completing learning phases (e.g. Foundations Master, UI/UX Craftsman, Server Wizard, Cloud Commander).

## Getting Started

### Prerequisites

Since this application is built entirely with vanilla web technologies, it runs directly in the browser and requires **zero dependencies or compilers**.

### Running Locally

To run the application locally, clone the repository and open `index.html` directly in your browser, or run a lightweight development server:

#### Option A: Open File Directly
Double-click `index.html` in your file explorer.

#### Option B: Using Python Server
```bash
python -m http.server 8000
```
Open [http://localhost:8000](http://localhost:8000) in your browser.

#### Option C: Using Node.js (npx serve)
```bash
npx serve .
```
Open [http://localhost:3000](http://localhost:3000) in your browser.
